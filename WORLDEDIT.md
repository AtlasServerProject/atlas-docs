# WorldEdit

WorldEdit foi instalado como ferramenta administrativa para acelerar construção, ajustes de mapas, arenas, estruturas de eventos e manutenção visual dos lobbys.

## Versão instalada

- Mod: `worldedit-mod-7.3.8.jar`
- Plataforma: Fabric
- Minecraft: `1.21` / `1.21.1`
- Origem: Modrinth
- SHA-512 validado:

```text
e039492df0b486e7ce76d0eaf8cb11eadad2e78220600b8498ab8eef4642a29e310a6bcb4378257553b3639c6d0bc1ebf0f73df3a7a677c6af6baf86716b0bc7
```

## Instalação no servidor

Arquivo ativo:

```text
/opt/atlas/server/fabric/mods/worldedit-mod-7.3.8.jar
```

O servidor foi reiniciado após a instalação e o log confirmou:

```text
WorldEdit for Fabric (version 7.3.8+6939-7d32b45) is loaded
```

## Uso recomendado

WorldEdit deve ser tratado como ferramenta de staff/construção, não como recurso de gameplay.

Usos previstos:

- ajustes finos nos lobbys;
- criação de arenas;
- preparação de áreas de eventos;
- correções emergenciais de mapa;
- importação e edição de estruturas.

## Cuidados

- Evitar operações grandes enquanto o Chunky estiver pré-gerando o Survival Emerald.
- Preferir seleções pequenas e incrementais.
- Fazer backup antes de alterações grandes em mapas.
- Usar em horários sem jogadores quando a edição for pesada.

## Cliente

WorldEdit está instalado no servidor. Jogadores comuns não precisam instalar WorldEdit no cliente para entrar.

Para builders/staff, instalar WorldEdit ou WorldEditCUI no cliente pode ajudar na visualização de seleções, mas não é obrigatório para o servidor funcionar.
