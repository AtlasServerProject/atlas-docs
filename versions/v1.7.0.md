# Atlas Core v1.7.0

Data da implantação: 2026-06-29

## Bloqueio de movimento

- Jogadores sem autenticação ficam presos à posição inicial da conexão.
- Movimento, salto, queda, knockback e veículos não deslocam o jogador.
- A câmera continua livre para rotação.
- Correção de posição executada pelo servidor apenas quando há deslocamento.
- Bloqueio aplicado também quando a sessão ainda não terminou de carregar.
- Liberação automática imediatamente após `/register` ou `/login`.

## Experiência do jogador

- Lembrete na action bar a cada 2 segundos.
- Jogadores sem conta recebem instrução para `/register`.
- Jogadores registrados recebem instrução para `/login`.

## Sessões

- Expiração de 12 horas agora é aplicada ao estado em memória.
- Sessão expirada volta automaticamente ao estado bloqueado.
- Âncoras são removidas no disconnect e shutdown.

## Banco de dados

- Nenhuma migration nova nesta release.

## Validação

- Compilação Java concluída com sucesso.
- Build Gradle concluído com sucesso.
- JAR v1.6.0 preservado em `/opt/atlas/server/backups/mod-versions/`.
- JAR v1.7.0 implantado como a única versão do Atlas Core em `mods`.
- Serviço `atlas` reiniciado e confirmado como ativo.
