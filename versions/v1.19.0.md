# Atlas Core v1.19.0

## Survival Emerald

- Criada a dimensão `atlas:survival_emerald`.
- Área definida em 6000 × 6000 blocos, equivalente a 375 × 375 chunks.
- Centro do mapa definido em `0, 0`, com raio de pré-geração de 3000 blocos.
- Chunky integrado para renderizar antecipadamente todo o mapa e evitar picos de geração durante o jogo.
- A tarefa é salva ao desligar o servidor e retomada automaticamente na inicialização.

## Navegação

- Adicionado `/lobby` para jogadores autenticados no Survival Emerald.
- O comando retorna o jogador ao spawn do Lobby Emerald.
- O seletor e o comando compartilham o mesmo serviço de teleporte.

## Operação

```bash
infra/scripts/atlas-cli pregenerate-survival start
infra/scripts/atlas-cli pregenerate-survival status
infra/scripts/atlas-cli pregenerate-survival pause
infra/scripts/atlas-cli pregenerate-survival continue
```
