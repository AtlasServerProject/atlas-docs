# Atlas Core v1.6.0

Data da implantação: 2026-06-29

## Registro

- Comando `/register <senha> <confirmacao>`.
- Confirmação obrigatória antes da criação da conta.
- Senha mínima de 8 caracteres.
- Limite máximo de 72 bytes UTF-8, respeitando o BCrypt.
- Conta autenticada imediatamente após registro bem-sucedido.

## Login

- Comando `/login <senha>`.
- Comparação exclusiva contra o hash persistido.
- Sessão autenticada com validade prevista de 12 horas.
- Atualização de último login e último IP.
- Mensagens específicas para conta inexistente, senha incorreta e sessão ausente.

## Segurança

- BCrypt com custo 12.
- Biblioteca `org.mindrot:jbcrypt:0.4` embutida no JAR.
- Senhas não são armazenadas ou registradas em texto puro pelo Atlas.
- Respostas dos comandos são privadas.
- Bloqueios de gameplay e limite de tentativas permanecem para próximas versões.

## Banco de dados

- Utiliza `auth_accounts.password_hash` e `player_sessions` existentes.
- Nenhuma migration nova nesta release.
