# Atlas Core v1.3.0

Data da implantação: 2026-06-28

## Nametag

- Prefixo e cor do maior cargo exibidos acima do jogador.
- Layout nativo em uma linha: `[CARGO] Nick`.
- Visibilidade configurada como permanente pelo scoreboard.
- TAB e nametag compartilham a mesma identidade visual.
- O username verdadeiro não é alterado.

## Sincronização em tempo real

- Verificação de mudanças de rank a cada 5 segundos.
- Alterações feitas pelo banco, site ou futuro painel não exigem reconexão.
- Apenas jogadores cuja identidade visual mudou são atualizados.
- Estados de jogadores offline são removidos da memória.

## Banco de dados

- Nenhuma migration nova nesta release.

## Validação

- Compilação Java concluída com sucesso.
- Build Gradle concluído com sucesso.
- JAR v1.2.1 preservado em `/opt/atlas/server/backups/mod-versions/`.
- JAR v1.3.0 implantado como a única versão do Atlas Core em `mods`.
- Serviço `atlas` reiniciado e confirmado como ativo.
