# Atlas Core v1.13.0

Data da implantação: 2026-07-01

## Proteção de construção

- Quebra de blocos bloqueada para jogadores comuns no Auth Lobby.
- Colocação de blocos bloqueada para jogadores comuns no Auth Lobby.
- Exceção concedida somente aos cargos `OWNER`, `DONO`, `ADMIN` e `ADM`.
- A autorização reutiliza a hierarquia central do `RankService`.
- Tentativas bloqueadas exibem aviso privado com cooldown de dois segundos.
- A proteção se limita ao Auth Lobby e não define regras para mundos futuros.
