# v1.26.3

## Comando `/dev`

- Adicionado `/dev`.
- Restrito a Dono/OWNER e ADM/ADMIN.
- Ao executar, o jogador recebe OP vanilla no servidor.
- Dono/ADM passam a poder usar `/op` e `/deop` mesmo estando no Auth Hub.

## Observação de segurança

O comando não libera OP para jogadores comuns. A validação continua baseada nos cargos globais do Atlas.

## Validação pendente

- Entrar como Dono ou ADM.
- Executar `/dev`.
- Confirmar mensagem de modo dev ativado.
- Executar `/op <jogador>` se necessário.
