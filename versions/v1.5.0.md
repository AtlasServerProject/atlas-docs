# Atlas Core v1.5.0

Data da implantação: 2026-06-29

## Fundação de autenticação

- `AuthModule` integrado ao ciclo modular do Atlas.
- `AuthRepository` isolando todo acesso a `auth_accounts` e `player_sessions`.
- `AuthService` responsável pelo ciclo de conta e sessão.
- `AuthSessionCache` concorrente para jogadores online.
- Modelos separados para conta, sessão persistente e estado em memória.
- Listener de conexão limitado a chamadas do serviço.

## Ciclo de sessão

- Uma sessão não autenticada é criada quando o jogador entra.
- A sessão é encerrada no disconnect ou shutdown do módulo.
- O serviço expõe operações para registro e autenticação das próximas versões.
- Sessões autenticadas possuem expiração prevista de 12 horas.

## Segurança

- Nenhuma senha é comparada ou registrada em logs.
- Nenhum jogador é bloqueado nesta versão de fundação.
- BCrypt e comandos serão adicionados em uma entrega posterior.

## Banco de dados

- Utiliza as migrations existentes `008_auth.sql` e `009_sessions.sql`.
- Nenhuma migration nova nesta release.

## Validação

- Compilação Java concluída com sucesso.
- Build Gradle concluído com sucesso.
- JAR v1.4.0 preservado em `/opt/atlas/server/backups/mod-versions/`.
- JAR v1.5.0 implantado como a única versão do Atlas Core em `mods`.
- Serviço `atlas` reiniciado e confirmado como ativo.
- `Authentication Manager` carregado sem erros do Atlas.
