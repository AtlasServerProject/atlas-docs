# v1.25.0

## Integração da economia com CobbleDollars

- Definido que a economia oficial do Atlas usa o `CobbleDollars` como fonte de saldo.
- Criada uma ponte segura entre Atlas Core e CobbleDollars.
- `/saldo` agora exibe o saldo atual do CobbleDollars.
- `/addmoney <valor>` agora adiciona CobbleDollars ao jogador.
- O banco legado `economy_accounts.balance` foi preservado sem ser apagado.

## Segurança operacional

- A integração não cria dependência rígida de compilação contra o CobbleDollars.
- Se o mod não estiver instalado ou mudar a API interna, o Atlas registra aviso e evita derrubar o servidor.

## Validação

- Build do `atlas-core` validado com sucesso.

  - executar `/addmoney 100`;
