# Versões do Atlas Core

Esta pasta contém o histórico detalhado de cada versão implantada do mod.

Cada release deve possuir um arquivo próprio no formato:

```text
vX.Y.Z.md
```

O registro deve informar:

- data da implantação;
- funcionalidades adicionadas;
- correções realizadas;
- alterações no banco;
- validações executadas;
- nome do artefato implantado.

O número da versão deve ser o mesmo definido em `atlas-core/gradle.properties`.

Os arquivos JAR não devem ser armazenados nesta pasta. Apenas o histórico da release pertence à documentação.
