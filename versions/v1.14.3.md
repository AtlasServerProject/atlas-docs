# Atlas Core v1.14.3

Data da implantação: 2026-07-02

## Hierarquia do TAB

- Substituída a ordenação baseada no ID do cargo por uma chave derivada da prioridade.
- Definida a ordem `Dono > ADM > MOD > SUP > VIPs > Players`.
- Jogadores ainda sem cargo são colocados abaixo do cargo Player.
- Alterações futuras de prioridade passam a atualizar a equipe visual do jogador.
- Prefixos, cores e nomes visíveis permanecem inalterados.
