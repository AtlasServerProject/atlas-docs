# Atlas Core v1.0.0

Data da implantação: 2026-06-28

## Fundação

- Bootstrap modular do Atlas.
- Integração com PostgreSQL.
- Player System e cache de jogadores online.
- Economia persistente.
- Sistema de ranks e permissões com wildcards.
- Comandos `/atlas`, `/saldo` e `/addmoney`.

## Identidade visual

- `RankFormatter` com cores nomeadas e hexadecimais.
- Prefixo do maior cargo exibido no TAB por equipes do scoreboard.
- Atualização visual no login.

## Administração de ranks

- Comando `/rank set <jogador> <cargo>`.
- Suporte a jogadores offline já cadastrados.
- Acesso restrito ao console, RCON, OWNER/DONO e ADMIN/ADM.
- Atualização imediata do TAB para jogadores online.

## Banco de dados

- Nenhuma migration nova nesta release.
- O campo `ranks.color` passou a ser carregado pelo Atlas Core.

## Validação

- Build Gradle concluído com sucesso.
- JAR implantado em `/opt/atlas/server/fabric/mods/atlas-core-1.0.0.jar`.
- Serviço `atlas` reiniciado e confirmado como ativo.
- Login de `VFSomente` validado com o nome formatado `[OWNER] VFSomente`.
