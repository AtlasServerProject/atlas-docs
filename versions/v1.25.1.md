# v1.25.1

## Pagamento entre jogadores

- Adicionado `/pay <player> <valor>`.
- O comando transfere CobbleDollars entre jogadores online.
- O jogador não pode pagar a si mesmo.
- O pagamento só conclui se houver saldo suficiente.
- Pagador e destinatário recebem confirmação no chat.
