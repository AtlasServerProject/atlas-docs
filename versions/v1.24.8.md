# v1.24.8

## Correção do fluxo Auth → Seletor

- Jogadores Premium não são mais teleportados automaticamente para a última posição do Survival Emerald ao entrar.
- `/login` e `/register` não restauram mais automaticamente a posição do Survival Emerald.
- Após autenticar, o jogador permanece no Auth Hub e deve escolher o destino pela bússola.
- A posição do Survival Emerald continua sendo salva quando o jogador está no Survival, mas não é usada para pular o seletor.

## Inventário real preservado no Auth Hub

- Ao entrar no Auth Hub, o inventário real do jogador é salvo temporariamente em memória.
- O Auth Hub limpa o inventário visual e entrega apenas os itens permitidos do hub.
- A bússola só é entregue quando o jogador está autenticado.
- Ao selecionar o Lobby Emerald pela bússola, o inventário real é restaurado antes do teleporte.
- Se o jogador desconectar ainda no Auth Hub, o inventário real também é restaurado antes da saída.
