# Atlas Core v1.23.2

## Demarcação visual das claims

- O primeiro canto selecionado com a pá dourada recebe um marcador temporário.
- Ao concluir uma claim, todo o perímetro é desenhado com blocos de ouro falsos.
- A inspeção posterior com graveto volta a exibir o perímetro.
- Os blocos são enviados somente ao cliente do jogador e não alteram o mapa.
- Não existem blocos reais, drops ou alterações persistentes.
- O terreno original reaparece automaticamente após 15 segundos.
- Claims grandes usam amostragem limitada para evitar excesso de pacotes.
