# Atlas Core v1.23.3

## Teleporte pelo `/claimslist`

- Cada claim aparece com o botão clicável `[TELEPORTAR]`.
- O cursor exibe uma mensagem explicando a ação.
- O servidor valida novamente o ID e o proprietário antes do teleporte.
- O destino é procurado dentro dos limites da claim no Survival Emerald.
- O terreno, fluidos, colisões, borda mundial e espaço para cabeça são validados.
- Claims removidas ou pertencentes a outro jogador não podem ser acessadas pelo comando interno.
