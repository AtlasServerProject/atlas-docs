# v1.26.0

## NPC Selector no Auth Lobby

- Adicionado o primeiro sistema nativo de NPCs do Atlas.
- Implementado NPC oficial `Lobby Emerald` no Auth Lobby.
- O NPC é invulnerável, silencioso, sem IA, com nome visível e respawn automático.
- Ao clicar sem autenticação, o jogador recebe aviso para fazer login.
- Ao clicar autenticado, o jogador é enviado ao Lobby Emerald pelo mesmo fluxo da bússola.
- O inventário real é preservado ao sair do Auth Lobby.

## Decisão de arquitetura

- O Auth Lobby continua sem gameplay.
- NPCs externos/legados continuam proibidos.
- A exceção são NPCs oficiais do Atlas usados apenas para navegação e orientação.

## Referência

O design foi inspirado no padrão de plugins como Citizens, mas implementado nativamente no Fabric/Atlas Core.
