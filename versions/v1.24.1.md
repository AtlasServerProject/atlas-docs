# Atlas Core v1.24.1

## Sprint 8 — Item Recovery

- `/lixeira` abre um inventário virtual com 54 espaços.
- Itens descartados permanecem recuperáveis durante 15 minutos.
- `/dropados` permite retirar total ou parcialmente os itens disponíveis.
- Itens não retirados são preservados ao fechar o menu.
- Drops antigos removidos automaticamente são recuperáveis quando possuem jogador proprietário.
- Serialização SNBT preserva todos os componentes do `ItemStack`.
- Persistência PostgreSQL pela migration `027_item_recovery.sql`.
- Expiração verificada a cada minuto.
- Armazenamento, abertura, recuperação e expiração possuem logs administrativos.
