# Atlas Core v1.1.0

Data da implantação: 2026-06-28

## Chat Formatter

- Layout compacto no formato `[CARGO] Nick » mensagem`.
- Exatamente um espaço entre cargo e nick.
- Remoção dos delimitadores `< >` do chat padrão.
- Cargo exibido na cor configurada no rank.
- Nick branco, separador cinza-escuro e mensagem cinza.
- Jogadores sem prefixo usam o formato `Nick » mensagem`.
- Mensagens de sistema, comandos, join/leave e mensagens privadas não são alteradas.

## Arquitetura

- Novo `ChatModule`.
- Formatação isolada em `ChatFormatter`.
- Broadcast concentrado em `ChatService`.
- Evento do Fabric limitado ao `ChatListener`.

## Banco de dados

- Nenhuma migration nova nesta release.

## Validação

- Compilação Java concluída com sucesso.
- Build Gradle concluído com sucesso.
- JAR v1.0.0 preservado em `/opt/atlas/server/backups/mod-versions/`.
- JAR v1.1.0 implantado em `/opt/atlas/server/fabric/mods/`.
- Serviço `atlas` reiniciado e confirmado como ativo.
- `ChatModule` carregado sem erros e inicialização do servidor concluída.
