# Atlas Core v1.4.0

Data da implantação: 2026-06-28

## Administração de ranks

- `/rank list`: lista cargos disponíveis e prioridades.
- `/rank info <jogador>`: mostra cargos ativos do jogador.
- `/rank remove <jogador> <cargo>`: desativa um vínculo preservando histórico.
- `/rank reload`: recarrega ranks, permissões e identidades visuais.
- `/rank permission add <cargo> <permissão>`: adiciona ou reativa uma permissão.
- `/rank permission remove <cargo> <permissão>`: desativa uma permissão.
- `/rank set` preservado para jogadores online e offline cadastrados.
- Acesso restrito ao console, Dono e Administrador.

## Banco de dados
